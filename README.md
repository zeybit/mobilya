# Mobilya Projesi

Bu proje, mobilya satış ve yönetim sistemi için geliştirilmiş bir web uygulamasıdır.

## Özellikler

- Ürün yönetimi
- Kullanıcı yönetimi
- Sipariş takibi
- Stok kontrolü

## Kurulum

1. Projeyi klonlayın:
```bash
git clone [repo-url]
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Gerekli ortam değişkenlerini ayarlayın:
```bash
cp .env.example .env
```

4. Uygulamayı başlatın:
```bash
npm start
```

## Teknolojiler

- Node.js
- Express.js
- MongoDB
- React.js
